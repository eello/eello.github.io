게리멘더링 2 풀이 - Kotlin

2026-02-26

문제 링크

조건 정리

  • 격자 크기: N(5 <= N <= 20)
  • 1 <= 각 구역의 인구 수 <= 100
  • 기준점 (r, c)
    • 왼쪽 상단 (0, 0) 기준
    • `0 <= r < N-2
    • 1 <= c < N-1
      • 0 <= c-d1
      • c+d2 < N
  • 경계 길이: 1 <= d1, d2
    • 1 <= d1 <= x
    • 1 <= d2 < N - x
    • x + d1 + d2 < N
  • 선거구 구역 - 왼쪽 상단 (0, 0), 기준점 (y, x) 기준
    • 1번 선거구: [0 <= r < y && 0 <= c <= x] + [y <= r < y+d1 && 0 <= c <= x-(1..d1)]
    • 2번 선거구: [0 <= r < y && x < c < N] + [y <= r <= y+d2 && x+(1..d2+1) < c < N]
    • 3번 선거구: [y+d1 <= r <= y+d1+d2 && 0 <= c < x-d1+(0..d2)] + [y+d1+d2 < r < N && 0 <= c < x-d1+d2]
    • 4번 선거구: [y+d1 < r <= y+d1+d2 && x+d2-(1..d2) < c < N] + [y+d1+d2 < r < N && x-d1+d2 <= c < N]
    • 5번 선거구: 그 외

예상 풀이

  • 유효한 모든 기준점 (r, c) 기준으로 반복: for (r in 0..<N-2) { for (c in 1..<N-2) {...} }
  • 모든 기준점마다 d1(1 <= d1 <= c), d2(1 <= d2 < N-c)의 모든 조합 구성: for (d1 in 1..<c) { for (d2 in 1..<N-c-1) { ... } }
  • 모든 조합마다 각 구역의 총 인원수 계산, 5구역은 (전체 인원수 - 나머지 4개 구역의 총 인원수)
    • 각 구역의 인원수 계산은 누적합 이용
  • 시간복잡도: N^2 + N^5
    • 누적합 계산(N^2) + 유효한 모든 기준점(N^2) * d1, d2 조합 구성(N^2) * 각 구역의 인원수 계산(누적합 이용: N)

누적합을 사용하는 이유

만일 각 구역의 인원수 계산이 누적합을 사용하지 않는다면, 2중 반복문으로 각 구역에 해당하는 모든 원소를 반복하며 인원수를 더해야하고 이 과정의 시간 복잡도는 O(N^2)
이렇게 각 구역의 인원수 계산에 N^2의 시간이 걸리게 되면 전체 시간 복잡도는 O(N^6)이 되고 N의 최대는 20으로 20^6 = 64,000,000번의 연산 횟수가 된다.

자바로 시간 제한 1초를 지키기 위해 1억 번 이하의 연산 횟수를 갖더라도 연산 자체가 가벼우면 통과될 가능성이 있긴 하지만 권장은 2000만~5000만 번 이하의 연산횟수를 갖는 것이 시간 제한에 여유롭다. (제미나이 추천 기준)
따라서 64,000,000번의 연산 횟수는 시간 제한에 걸릴 수 있어 줄이는 방법을 찾게 되었고 생각한 방법이 누적합이다.

최초 1회로 횡으로 누적합을 계산해 놓으면 전체 구역의 인구수를 구하는데 N의 시간복잡도만 필요하다. 누적합 계산은 N^2이 걸리지만 별도로 진행되기 때문에 최대 400번의 연산 횟수만 거치면 되고 기준점과 d1,d2 조합에 따른 각 구역의 인원수를 구하는 것에는 N^5이 걸리게 된다. 전체 시간복잡도는 O(N^2 + N^5)로 최대 400 + 20^5(3,200,000) = 3,200,400번의 연산 횟수가 필요하고 N^6보다 약 6천만번의 연산횟수를 줄일 수 있어 시간 제한에 여유로울 것으로 예상한다.

풀이

누적합을 계산하기 위한 함수를 만들어 횡(→) 방향의 누적합 배열을 리턴하는 함수를 정의해 가장 먼저 누적합을 계산하도록 했다.

private fun accumulate(arr: Array<IntArray>): Array<IntArray> {  
    val n = arr.size  
    return Array(n) { r ->  
        var acc = 0  
        IntArray(n) { c ->  
            acc += arr[r][c]  
            acc  
        }  
    }  
}

그리고 1-4 구역의 범위에 따른 총 인원수를 계산하기 위해 4개의 함수를 정의했다.
함수는 누적합 배열(acc)를 입력으로 받아 범위에 따라 누적합을 이용해 구역의 총 인원 수를 계산한다. 5구역은 도시의 전체 인원수에서 1-4 구역의 총 인원수를 빼면 구할 수 있기 때문에 함수로 정의하지 않았다.

// [0 <= r < y && 0 <= c <= x] + [y <= r < y+d1 && 0 <= c <= x-(1..d1)]  
private fun getSection1(acc: Array<IntArray>, y: Int, x: Int, d1: Int, d2: Int): Int {  
    var sum = 0  
    var temp = 0  
    for (r in 0..<y + d1) {  
        if (y <= r) {  
            temp++  
        }  
  
        sum += acc[r][x - temp]  
    }  
  
    return sum  
}  
  
// [0 <= r < y && x < c < N] + [y <= r <= y+d2 && x+(1..d2+1) < c < N]  
private fun getSection2(acc: Array<IntArray>, y: Int, x: Int, d1: Int, d2: Int): Int {  
    val lastIndex = acc[0].lastIndex  
    var sum = 0  
    var temp = 0  
  
    for (r in 0..<y + d2 + 1) {  
        sum += acc[r][lastIndex] - acc[r][x + temp]  
        if (y <= r) {  
            temp++  
        }  
    }  
  
    return sum  
}  
  
// [y+d1 <= r <= y+d1+d2 && 0 <= c < x-d1+(0..d2)] + [y+d1+d2 < r < N && 0 <= c < x-d1+d2]  
private fun getSection3(acc: Array<IntArray>, y: Int, x: Int, d1: Int, d2: Int): Int {  
    val lastIndex = acc.lastIndex  
    var sum = 0  
    var temp = 0  
  
    for (r in y + d1..lastIndex) {  
        if (0 <= x - d1 - 1 + temp) {  
            sum += acc[r][x - d1 - 1 + temp]  
        }  
  
        if (r < y + d1 + d2) {  
            temp++  
        }  
    }  
  
    return sum  
}  
  
// [y+d2 < r <= y+d1+d2 && x+d2-(1..d2) < c < N] + [y+d1+d2 < r < N && x-d1+d2 <= c < N]  
private fun getSection4(acc: Array<IntArray>, y: Int, x: Int, d1: Int, d2: Int): Int {  
    val lastIndex = acc.lastIndex  
    var sum = 0  
    var temp = 0  
  
    for (r in y + d2 + 1..lastIndex) {  
        sum += acc[r][lastIndex] - acc[r][x - temp + d2 - 1]  
        if (r <= y + d1 + d2) {  
            temp++  
        }  
    }  
  
    return sum  
}

각 구역의 인원수의 최고값과 최소값의 차이를 구하는 함수 getMinDiff()를 정의했다. 여기서 가능한 모든 기준점 (y, x)와 (d1, d2) 조합을 만들기 위해 4중 반복문을 사용했다.

  • funs는 1-4 구역의 인원수를 계산하는 함수가 담긴 리스트로 조합에 따른 각 구역의 인원수를 계산하는 코드를 구현하기 쉽게 만들기 위해 추가, 이를 위해 각 함수(getSection1-4)에 사용되지 인자라도 정의에 추가해 통일
private fun getMinDiff(arr: Array<IntArray>): Int {  
    val n = arr.size  
    val acc = accumulate(arr)  
    val total = acc.sumOf { it[it.lastIndex] }  
    val funs = arrayOf<(Array<IntArray>, Int, Int, Int, Int) -> Int>(  
        ::getSection1, ::getSection2, ::getSection3, ::getSection4  
    )  
  
    var answer = total  
    for (y in 0..<n - 2) {  
        for (x in 1..<n - 1) {  
            for (d1 in 1..x) {  
                for (d2 in 1..<n - x) {  
                    if (y + d1 + d2 >= n) {  
                        continue  
                    }  
  
                    var max = 0  
                    var min = total  
                    var s5 = total // 5구역  
                    for (f in funs) { // 1~4구역 계산  
                        val p = f(acc, y, x, d1, d2)  
                        max = max(max, p)  
                        min = min(min, p)  
                        s5 -= p  
                    }  
  
                    max = max(max, s5)  
                    min = min(min, s5)  
  
                    answer = min(answer, max - min)  
                }  
            }  
        }  
    }  
  
    return answer  
}
